Tema 2. Define los elementos que componen la estructuración de una página web:

1. Contenedor (Wrapper)

Es un elemento que engloba toda la estructura de la página. Sirve para limitar el ancho del contenido y centrarlo en la página. Se utiliza principalmente para aplicar estilos y gestionar el diseño de manera uniforme.

2. Cabecera (Header)

Es la parte superior de la página, donde generalmente se incluye el título del sitio, el logotipo, y enlaces importantes como el menú de navegación. Puede contener elementos como un buscador o enlaces a redes sociales.

3. Contenido (Content)

Es la sección principal donde se muestra la información relevante del sitio. Aquí se puede incluir texto, imágenes, videos y otros elementos multimedia. Es la parte más importante desde el punto de vista del usuario.

4. Menú (Menu)

Es un conjunto de enlaces que permite a los usuarios navegar por las distintas secciones del sitio web. Puede ser horizontal o vertical y generalmente se encuentra en la cabecera o en la barra lateral.

5. Pie (Footer)

Es la parte inferior de la página. Suele contener información adicional como derechos de autor, enlaces de contacto, política de privacidad, y enlaces a redes sociales. Es una sección importante para ofrecer información complementaria.

6. Lateral (Sidebar)

Es una sección que se encuentra a un lado del contenido principal. Suele contener enlaces adicionales, widgets, o información secundaria. Suelen usarse para mostrar menús, anuncios o enlaces a artículos populares.

7. Etiqueta div

Es un elemento de bloque en HTML que se utiliza para agrupar otros elementos y aplicar estilos CSS. No tiene un significado semántico específico, pero es muy útil para estructurar el contenido y facilitar el diseño.

Tema 3. Identifica y describe los elementos que componen la estructuración semántica de una página web.

header

Define la cabecera de una sección o de la página completa. Suele contener el título, logotipos, y elementos de navegación. Ayuda a organizar el contenido de manera que los usuarios y los motores de búsqueda comprendan su importancia.

nav

Especifica un bloque de navegación. Contiene enlaces a otras páginas o secciones dentro del mismo sitio. Utilizar nav mejora la accesibilidad y la comprensión del contenido por parte de los motores de búsqued

main

Representa el contenido principal del documento. Debe contener el contenido que es único a la página y que no se repite en otras secciones, como el encabezado y el pie de página.

aricle

Define un contenido independiente y autocontenido que podría distribuirse de manera independiente (como un artículo de un blog, una noticia o una entrada de foro). Ayuda a los motores de búsqueda a identificar contenido relevante.

section

Agrupa contenido relacionado temáticamente. Cada sección puede tener su propio encabezado y puede ser usada para dividir el contenido en partes más manejables. Es útil para la organización del contenido.

aside

Representa contenido relacionado de manera indirecta con el contenido principal, como barras laterales, notas, o información adicional. Ayuda a proporcionar contexto sin desviar la atención del contenido principal.

footer

Define el pie de una sección o de la página. Suele incluir información sobre el autor, derechos de autor, enlaces de contacto y otras referencias. Proporciona un cierre lógico a la información presentada.

h1 a h6

Estas etiquetas definen los encabezados en orden jerárquico. es el más importante y se usa generalmente para el título principal, mientras que se utilizan para subtítulos y secciones dentro del contenido.

figure & figcaption

La etiqueta figure se usa para encapsular contenido gráfico como imágenes, gráficos o ilustraciones, mientras que figcaption proporciona una leyenda o descripción para ese contenido.

time

Se utiliza para representar fechas y horas. Facilita que los motores de búsqueda y las aplicaciones comprendan la temporalidad de un evento o contenido.

address

Especifica información de contacto del autor o propietario del documento. Es útil para proporcionar un punto de referencia para los usuarios
